## Source Code

The source code for this project is hosted on GitHub. You can find the
repository [here](https://github.com/morethanwords/tweb).

## Cloning the Repository

To get started, you'll need to clone the repository to your local machine. Open
your terminal and run the following command:

```bash
$ git clone git@github.com:morethanwords/tweb.git
```

This command will clone the repository to your local machine, providing you with
the latest version of the source code.

## Installing Dependencies

This project requires Node.js version 18 or higher. Make sure you have Node.js
installed on your machine. You can download it from
[here](https://nodejs.org/en).

Additionally, the project uses [pnpm](https://pnpm.io/) as its package manager.
If you haven't installed pnpm yet, you can do so globally using:

```bash
$ npm install -g pnpm
```

After installing pnpm, navigate to the project's directory:

```bash
$ cd tweb
```

Now, install the project dependencies:

```bash
$ pnpm install
```

This command will download and install all the required dependencies for the
project.

## Starting the Project

Once the dependencies are installed, you can start the project using either npm
or pnpm. Run one of the following commands:

```bash
$ npm start # pnpm start
```

This will initiate the project and you should be able to access it in your web
browser at http://localhost:8080.
